## Calmly – Minimalist Meditation + Sleep Sounds App
A clean, minimal Android app for meditation and sleep, designed to help users relax with soothing ambient sounds and sleep noises. Built as an internship assignment with a focus on modern Android practices, media playback, and an immersive user experience.

## 🎯 Objective
To develop a fully functional Android application that enables users to:

-Play calming sounds from different categories.

-Control playback using an intuitive UI and persistent notification controls.

-Maintain a clean, distraction-free experience for both meditation and sleep.

## ✅ Features
### 🧘 Meditation Tab
#### Includes 4–5 ambient sound categories:
Forest, Rain, Campfire, Ocean

### 🌙 Sleep Tab
#### Includes 4–5 sleep sound categories:

White Noise,  Lullaby, Fan, Deep Hum

## ⏯️ Media Playback
-Looped Playback: All sounds loop seamlessly.

-Single Sound Policy: Only one sound plays at a time.

-Persistent Foreground Media Service.

-Media Controls in Notification: Play/Pause and Stop actions supported.

🛠️ Technical Stack
-Language: Kotlin

-Architecture: MVVM

-Jetpack Components: ViewModel, LiveData

-Media Playback: Foreground Media Service

-Notification: Custom media notification with persistent controls
